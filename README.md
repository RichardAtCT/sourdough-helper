# Sourdough Helper

An interactive web application to help beginners make sourdough bread with confidence. Features built-in timers, recipe scaling, baker's percentages, and step-by-step guidance.

## Features

- **Recipe Scaling**: Easily scale the recipe from 0.5x to 3x
- **Interactive Timers**: Multiple simultaneous timers for each stage of bread making
- **Baker's Percentages**: Toggle to show professional baker's percentages
- **Step Tracking**: Check off each step as you complete it
- **Timeline Calculator**: See a suggested timeline based on your start time
- **Mobile Friendly**: Responsive design works on all devices

## Live Demo

Visit the live application: [Sourdough Helper](https://[your-username].github.io/sourdough-helper/)

## Usage

Simply open the page and follow along:

1. Adjust the recipe scale if needed (default is 1x for 500g flour)
2. Optionally set your start time to see a suggested timeline
3. Follow the step-by-step instructions
4. Use the built-in timers for each stage
5. Check off steps as you complete them

## Recipe

The base recipe (at 1x scale):
- 500g All-purpose flour
- 105g Active starter (21%)
- 342g Water (68.4%)
- 11g Salt (2.2%)

## Technology

Built as a single-page application using:
- React 18
- Tailwind CSS
- Lucide Icons

No build process required - the entire app is self-contained in a single HTML file.

## Local Development

To run locally, simply open `index.html` in any modern web browser.
